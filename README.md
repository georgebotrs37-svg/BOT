🔴 Python Website ChatBot

A simple Website ChatBot built with Python using the Flask framework.
This bot runs on a local web server and allows users to chat with a simple automated response system.

---

🚀 Features

- Simple Website Chat Interface
- Built with Python and Flask
- Real-time message interaction
- Easy to modify bot responses
- Modern UI with red theme

---

🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript (jQuery)

⚙️ Installation

1. Clone the repository

git clone https://github.com/georgebotrs37-svg/python-chatbot.git

2. Go to the project folder

cd python-chatbot

3. Install requirements

pip install flask

---

▶️ Run the Bot

Run the following command:

python app.py

Then open your browser and go to:

http://127.0.0.1:5000
